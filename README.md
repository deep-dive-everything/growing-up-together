## 📌 진행 방식 
<div align="left">

- 스터디는 매주 일요일 20시 30분에 진행됩니다.
- 매주 정해진 분량 만큼 책을 읽은 후 내용 요약 및 느낀점을 정리하여 업로드합니다.
- 발표자의 진행에 따라 각 주제에 대한 자신의 생각을 공유합니다.
